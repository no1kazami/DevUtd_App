# DevUtd_App
<img src="자료/UI/Etc/DevUtd_Screenshot.jpg"  width="216" height="468"/>
2020년 창단한 아마추어 축구팀 데브 유나이티드 FC의 공식 어플


## 🖥️ 프로젝트 소개
제가 활동하고 있는 아마추어 축구팀 데브 유나이티드 FC의 경기 일정, 여러가지 기록 정보등을 팀원들과 공유하기 위해 개발한 어플입니다. 
<br>

## 🕰️ 개발 기간
* 21년 11월 ~ 21년 12월
* 개발 이후 필요할때마다 업데이트 중

## ⚙️ 개발 환경
- `Visual Studio 2019`
- `Firebase`
- `UE4`
